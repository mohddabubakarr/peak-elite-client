# PEAK ELITE W.L.L Website

Brand-new client website for PEAK ELITE W.L.L, a Bahrain-based facility support and hospitality services company.

## Scope

- Cleaning services
- Facility maintenance support
- Hospitality staffing support
- Event support services
- General operational support
- Lead-focused contact section for Bahrain market enquiries

## Development

```bash
npm install
npm run dev
```
