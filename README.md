# PEAK ELITE W.L.L Website

Brand-new client website for PEAK ELITE W.L.L, a Bahrain-based facility support and hospitality services company.

## Scope

- Cleaning services
- Facility maintenance support
- Hospitality staffing support
- Event support services
- Home & wardrobe organisation
- General operational & retail support
- Lead-focused contact section for Bahrain market enquiries

## SEO

- `index.html` carries the primary meta description, Open Graph/Twitter tags, geo tags, canonical URL, and `LocalBusiness` + `FAQPage` JSON-LD structured data.
- `public/robots.txt` and `public/sitemap.xml` are served at the site root for search engine crawling.
- `public/google12016654ee0310c5.html` is the Google Search Console ownership-verification file — do not remove it.
- Service and FAQ copy in `src/content.js` should stay aligned with the structured data in `index.html` if either changes.
